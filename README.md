# FocusTube 🎯

FocusTube is a Chrome extension designed to reduce distractions on YouTube and improve productivity for students, developers, and learners.

---

## 🚀 Features

✅ Hide YouTube Shorts
✅ Hide comments
✅ Hide recommendations
✅ Study Mode (Educational videos only)
✅ Educational vs Entertainment analytics
✅ Focus score tracking
✅ Daily study streak
✅ 25-minute Pomodoro focus timer
✅ Modern animated UI

---

## 🧠 Study Mode

FocusTube filters YouTube content and allows only educational videos based on keywords like:

* Coding
* Programming
* Java
* Python
* AI
* Design
* Tutorials

Entertainment and distracting videos are automatically hidden.

## ⚡ Installation

1. Download this repository
2. Open Chrome browser
3. Go to:

```txt
chrome://extensions
```

4. Enable Developer Mode
5. Click "Load unpacked"
6. Select project folder
